# singed_slices

## How to run
- cd into script directory
- run `python3 render.py`
- enjoy :\)

## Controls
#### Zoom in/out
- mouse scroll up/down
#### Rotate
- click and drag mouse
#### Slide (left/right/up/down)
- arrow keys
#### Play/Pause Simulation
- space bar
